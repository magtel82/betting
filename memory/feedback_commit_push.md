---
name: Commit och push alltid
description: Användaren vill att alla kodändringar alltid committas och pushes direkt — inga separata frågor om det
type: feedback
---

Committa och pusha alltid direkt efter att ändringar är gjorda, utan att fråga om det.

**Why:** Användaren vill inte behöva påminna om detta varje gång.

**How to apply:** Efter varje session med kodändringar — kör `git add`, `git commit` och `git push` automatiskt som sista steg.
